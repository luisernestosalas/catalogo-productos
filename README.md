# Catálogo de Productos 📦

Una aplicación web sencilla y funcional para gestionar un catálogo de productos con información de precios y stock en tiempo real.

## 🌐 Demo en Vivo

**[Ver Aplicación](https://catalogo-productos-ijarj8wl1-luisernestosalas-2775s-projects.vercel.app)**

## 📋 Características

- ✅ **Visualización de productos** con imagen, nombre, precio y stock
- ✅ **Interfaz responsive** que se adapta a cualquier dispositivo
- ✅ **Búsqueda y filtrado** de productos
- ✅ **Gestión de stock** con indicadores visuales
- ✅ **Diseño limpio y moderno**
- ✅ **Carga rápida** sin dependencias externas

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos y diseño responsive
- **JavaScript** - Lógica de la aplicación e interactividad
- **Vercel** - Deployment y hosting
