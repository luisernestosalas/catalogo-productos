## Descripción del Proyecto

Aplicación web frontend que implementa un catálogo de productos con funcionalidades de visualización de precios y control de stock. Desarrollada con tecnologías web básicas para máxima compatibilidad y rendimiento.

## Arquitectura Técnica

- **Deployment:** Vercel
