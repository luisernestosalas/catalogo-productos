# Manual de Usuario - Catálogo de Productos

## ¿Qué es esta aplicación?

Es una aplicación web simple que te permite ver productos con sus precios y cantidades disponibles. Puedes buscar productos y ver toda la información importante de cada uno.

## ¿Cómo acceder?

Entra a la página web desde tu navegador:
**https://catalogo-productos-ijarj8wl1-luisernestosalas-2775s-projects.vercel.app**

## Funciones principales

### 1. Ver todos los productos
- Al entrar, verás una lista con todos los productos disponibles
- Cada producto muestra:
  - Nombre del producto
  - Precio
  - Cantidad en stock (cuántos quedan)

### 2. Buscar productos
- Usa la caja de búsqueda en la parte superior
- Escribe el nombre del producto que buscas
- Los resultados aparecen automáticamente mientras escribes

### 3. Información del stock
- **Verde**: Hay productos disponibles
- **Rojo**: Sin stock (agotado)
- **Número**: Cantidad exacta disponible

## Cómo usar la aplicación

1. **Abrir la página**: Entra al enlace desde cualquier navegador
2. **Ver productos**: Desplázate por la lista para ver todos los productos
3. **Buscar**: Si buscas algo específ
